# NEURO COMMENTING

## Руководство пользователя

Это главное руководство по работе в Telegram-боте. Оно написано для оператора сервиса и показывает только реальный путь в интерфейсе:

`прокси -> аккаунты -> восстановление -> подготовка -> каналы -> запуск`

## С чего начать

Откройте бот и пройдите шаги по порядку:

1. `🎯 Продукт`
2. `🌐 Прокси`
3. `👤 Аккаунты`
4. `📢 Каналы`
5. `🔍 Поиск каналов`
6. `💬 Автоматический режим`

Если идти именно в таком порядке, бот будет показывать понятные следующие действия на каждом этапе.

Если хотите полностью начать заново и убрать старые аккаунты из рабочего контура, сначала откройте:

`👤 Аккаунты -> Начать с нуля`

Бот очистит аккаунты, каналы, кандидаты поиска и очереди, но сохранит логи, quarantine и архив snapshot.

## 1. Продукт

Откройте:

`🎯 Продукт`

Здесь проверьте:

- название продукта;
- ссылку на бот или сервис;
- общий темп работы;
- баланс нейтральных и более прямых сообщений.

Если ссылка на продукт изменилась, сначала обновите её здесь, а потом переходите к аккаунтам и каналам.

## 2. Прокси

Откройте:

`🌐 Прокси -> Загрузить из файла`

Что нужно сделать:

- загрузить список прокси;
- убедиться, что прокси появились в списке;
- только после этого переходить к загрузке аккаунтов.

Важно:

- для каждого аккаунта нужен свой прокси;
- перед проверкой доступа бот сам проверяет, жив ли прокси и не занят ли он другим аккаунтом;
- нерабочие прокси отключаются автоматически;
- если свободных прокси становится мало, бот заранее предупреждает, что пора загрузить новые;
- если список прокси обновился, после этого снова откройте `👤 Аккаунты -> Проверить доступ` и выберите нужный номер.

## 3. Как добавить аккаунты

Откройте:

`👤 Аккаунты -> Загрузить аккаунт`

Загрузите в чат:

- `phone_number.session`
- `phone_number.json`

Порядок загрузки не важен. Бот сам собирает комплект из двух файлов и показывает:

- комплект готов;
- ждёт `.json`;
- ждёт `.session`.

После загрузки:

1. откройте `👤 Аккаунты -> Начать настройку аккаунта`
2. выберите номер, который хотите вести шаг за шагом
3. затем откройте `👤 Аккаунты -> Проверить доступ`
4. выберите этот же номер

Что должно получиться:

- аккаунт появился в разделе `📋 Мои аккаунты`;
- комплект файлов готов;
- бот показывает, что доступ есть;
- аккаунт появился в пошаговой настройке;
- аккаунт можно отправить на подготовку.

## 4. Как восстановить доступ

Если бот показывает, что с аккаунтом есть проблема, откройте:

`👤 Аккаунты -> Восстановить аккаунты`

Там для каждого номера будет видно:

- текущее состояние;
- что нужно сделать дальше;
- готов ли аккаунт к подготовке;
- нужен ли повторный вход.

Основные действия:

- `Загрузить заново` — если нужно заменить файлы аккаунта;
- `Проверить доступ ещё раз` — если вы уже обновили аккаунт;
- `Сгенерировать черновик профиля` — если доступ уже подтверждён;
- `Продолжить настройку` — если черновик уже создан и нужно двигаться дальше.

Если бот пишет `Нужен повторный вход`, сначала загрузите свежие файлы аккаунта, потом снова нажмите `Проверить доступ` и выберите этот номер.

Пока комплект не собран и вы не запустили `Проверить доступ`, аккаунт в работу не идёт.

Если бот пишет `Ограничен`, сначала проверьте аккаунт вручную, а потом возвращайтесь в бот и снова запускайте проверку.

## 5. Пошаговая настройка аккаунта

Для одного аккаунта используйте экран:

`👤 Аккаунты -> Продолжить настройку`

Там видно:

- текущий этап;
- последний результат проверки;
- что делать дальше;
- последние сохранённые шаги.

В карточке аккаунта доступны основные действия:

- `Проверить доступ`
- `Сгенерировать черновик профиля`
- `Подтвердить профиль`
- `Сгенерировать черновик канала`
- `Подтвердить канал`
- `Сгенерировать черновик поста`
- `Подтвердить пост`
- `Назначить роль аккаунта`
- `История шагов`

Каждое действие сохраняется в историю настройки.

## 6. Как подготовить аккаунт

Подготовка теперь идёт через черновики Gemini и ручное подтверждение каждого шага.

Путь такой:

1. `👤 Аккаунты -> Черновики и применение`
2. `👤 Аккаунты -> Продолжить настройку`
3. `Сгенерировать черновик профиля`
4. `Подтвердить профиль`
5. `Сгенерировать черновик канала`
6. `Подтвердить канал`
7. `Сгенерировать черновик поста`
8. `Подтвердить пост`
9. `Назначить роль аккаунта`

Во время подготовки бот:

- сначала показывает черновик;
- не применяет Telegram-действия без вашего подтверждения;
- сохраняет историю каждого шага;
- после поста даёт выбрать роль аккаунта.

Если на любом apply-шаге бот показывает ограничение, остановитесь и вернитесь в экран восстановления аккаунтов.

## 7. Каналы

Если у вас уже есть свои каналы, добавляйте их так:

`📢 Каналы -> Добавить канал`

После добавления канал появится в базе и будет учитываться в общей работе.

В разделе `📢 Каналы` вы можете:

- посмотреть всю базу;
- открыть список найденных каналов;
- посмотреть статистику;
- исключить канал из работы.

## 8. Как искать новые каналы

Откройте:

`🔍 Поиск каналов`

Там доступны:

- поиск по ключевым словам;
- поиск по тематике;
- поиск похожих каналов;
- фильтры по подписчикам и активности.

Найденные каналы не идут в работу сразу. Сначала они попадают в список проверки:

`📢 Каналы -> Проверить найденные каналы`

Для каждого канала можно выбрать:

- `Оставить для проверки`
- `Разрешить автоматически`
- `Только вручную`
- `Не использовать`

Если хотите отправить текущую сводку найденных и готовых каналов в отдельного бота, используйте:

`🔍 Поиск каналов -> Отправить сводку`

## 9. Как включить автоматический режим

Когда:

- аккаунты готовы к работе;
- нужные каналы отмечены для работы;
- проблемные аккаунты восстановлены;

откройте:

`💬 Автоматический режим -> Управление автопилотом`

Дальше:

1. нажмите `▶️ Включить автоматический режим`
2. откройте `📊 Что происходит сейчас`
3. при необходимости смотрите `📝 История комментариев`
4. перед запуском новых сценариев используйте `🧪 Проверить текст`

Если бот пишет, что автоматический режим недоступен, вернитесь в `Аккаунты` и `Каналы` и завершите недостающие шаги.

## Что значат статусы

`Новый аккаунт`

- аккаунт загружен, но ещё не подготовлен.

`Идёт подготовка`

- бот оформляет профиль и готовит аккаунт к работе.

`Прогрев`

- аккаунт проходит последние шаги перед запуском.

`Ждёт подтверждения`

- подготовка почти закончена, осталось финальное подтверждение.

`Готов к работе`

- аккаунт можно использовать в автоматическом режиме.

`Нужен повторный вход`

- файлы аккаунта нужно обновить и снова проверить доступ.

`Ограничен`

- аккаунт требует ручной проверки перед продолжением.

`Нужна перепроверка`

- подготовка остановилась, и аккаунт нужно проверить ещё раз.

## Короткий маршрут на каждый день

Если вы заходите в бот каждый день, обычно достаточно такого порядка:

1. `📊 Обзор`
2. `👤 Аккаунты -> Восстановить аккаунты`
3. `📢 Каналы -> Проверить найденные каналы`
4. `💬 Автоматический режим -> Что происходит сейчас`

Если всё в порядке, вмешательство не требуется.
