# Account Onboarding Runbook

Эталонный путь настройки одного аккаунта:

1. Загрузить пару файлов `.session + .json`.
2. Начать настройку аккаунта.
3. Проверить доступ только для этого номера.
4. Если доступ есть, поставить аккаунт в подготовку.
5. После подготовки отметить шаги:
   - профиль готов;
   - канал готов;
   - подписки готовы.
6. Если аккаунт дошёл до `Ждёт подтверждения`, открыть `Завершить подготовку`.
7. После подтверждения аккаунт считается готовым.

Каждый шаг должен сохраняться:

- в таблицы onboarding runs / steps;
- в историю lifecycle;
- в `account_onboarding_memory.md`.
