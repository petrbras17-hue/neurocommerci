# БЫСТРЫЙ СТАРТ — Новые аккаунты

## Что покупать

Telegram аккаунты с **активными .session файлами** (формат Telethon/Pyrogram).
Каждый аккаунт = `.session` файл + `.json` с метаданными.

**Важно при покупке:**
- Формат: Telethon `.session` файлы
- С 2FA паролем (облачный пароль)
- Русские номера (+7) предпочтительно
- Не подключать больше 1 аккаунта с одного IP!

## Шаг 1: Подготовка

### Положить файлы
Скопировать `.session` и `.json` файлы в:
```
data/sessions/
├── 79XXXXXXXXX.session
├── 79XXXXXXXXX.json
├── 79YYYYYYYYY.session
├── 79YYYYYYYYY.json
└── ...
```

### Прокси
Положить рабочий прокси в `data/proxies.txt`:
```
host:port:user:pass
```
Один ротационный прокси подходит для всех аккаунтов (каждое подключение = новый IP).

### .env настройки
Проверить что в `.env`:
```
PROXY_TYPE=http          # или socks5, зависит от прокси
PROXY_ROTATING=false     # true если прокси ротационный
```

## Шаг 2: Запуск

```bash
cd "NEURO COMMENTING"
source venv/bin/activate
python run_full_setup.py
```

Скрипт автоматически:
1. Найдёт все `.session` файлы
2. Зарегистрирует аккаунты в БД
3. Подключит каждый через прокси (с задержками 5с)
4. Упакует профили (женские имена, аватарки, username)
5. Создаст каналы-переходники с рекламой DartVPN

## Шаг 3: Запуск бота

```bash
python main.py
```

В Telegram боте (`@NeuroCommentingBot`):
- **Аккаунты** → Список → все аккаунты видны
- **Аккаунты** → Подключить все → подключение через прокси
- **Аккаунты** → Упаковка → установка профилей
- **Аккаунты** → Каналы → создание каналов-переходников

## Правила антибана (ВАЖНО!)

1. **НИКОГДА** не подключать больше 1 аккаунта с одного IP без прокси
2. Задержки между действиями: 15-30 секунд между аккаунтами
3. Прогрев: 5 комментов/день → 10 → 20 → 35 (за 4 дня)
4. Один прокси = один аккаунт (или ротационный прокси для всех)
5. Не менять профиль сразу после покупки — подождать 1-2 часа

## Если сессии отозвали

Сессии = одноразовый ключ. Если Telegram отозвал:
- Восстановить НЕВОЗМОЖНО без SMS-кода
- SMS-коды приходят только владельцу номера (продавцу)
- Нужно покупать НОВЫЕ аккаунты

**Главная причина отзыва:** несколько аккаунтов с одного IP.
