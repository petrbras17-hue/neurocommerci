# QUICK START

Главное руководство пользователя: [USER_GUIDE.md](/Users/braslavskii/NEURO%20COMMENTING/USER_GUIDE.md)

Если нужен только короткий маршрут, идите так:

1. `👤 Аккаунты -> Начать с нуля`
Если хотите стартовать заново, сначала очистите старое user-state.
Логи, quarantine и архив snapshot сохранятся.

2. `🎯 Продукт`
Проверьте название и ссылку на продукт.

3. `🌐 Прокси -> Загрузить из файла`
Сначала загрузите прокси для аккаунтов.

4. `👤 Аккаунты -> Загрузить аккаунт`
Добавьте в бот оба файла аккаунта: `.session` и `.json`.

5. `👤 Аккаунты -> Начать настройку аккаунта`
Выберите номер, который хотите вести шаг за шагом.

6. `👤 Аккаунты -> Проверить доступ`
Выберите этот же номер и дождитесь проверки только по нему.
Перед проверкой бот сам перепроверит и закрепит за аккаунтом один живой прокси.

7. `👤 Аккаунты -> Обновить список`
Убедитесь, что аккаунт появился в системе и его статус обновился.

8. `👤 Аккаунты -> Восстановить аккаунты`
Если есть проблемы, бот сам покажет, что делать дальше.

9. `👤 Аккаунты -> Черновики и применение`
Откройте новый human-gated путь настройки.

10. `👤 Аккаунты -> Продолжить настройку`
Откройте карточку аккаунта и идите по шагам:
`Сгенерировать черновик профиля -> Подтвердить профиль -> Сгенерировать черновик канала -> Подтвердить канал -> Сгенерировать черновик поста -> Подтвердить пост`.

11. `👤 Аккаунты -> Продолжить настройку -> Назначить роль аккаунта`
После поста выберите роль аккаунта и переведите его в рабочий пул.

12. `📢 Каналы -> Добавить канал`
Добавьте свои каналы.

13. `🔍 Поиск каналов`
Найдите новые подходящие каналы.

14. `📢 Каналы -> Проверить найденные каналы`
Отметьте, какие каналы готовы к работе.

15. `🔍 Поиск каналов -> Отправить сводку`
Отправьте итоговую сводку найденных каналов в отдельного digest-бота.

16. `💬 Автоматический режим -> Управление автопилотом`
Включите автоматический режим.

17. `📊 Обзор`
Проверьте, сколько аккаунтов и каналов готовы к работе.
Если запас свободных прокси падает, бот покажет предупреждение заранее.
